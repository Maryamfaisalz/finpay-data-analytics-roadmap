# Data Dictionary

## customers.csv

Contains customer demographic and financial information.

| Column | Description |
|--------|-------------|
| customer_id | Unique identifier for each customer |
| first_name | Customer's first name |
| last_name | Customer's last name |
| gender | Male or Female |
| age | Customer age |
| city | Customer city |
| country | Customer country |
| signup_date | Date the customer joined FinPay |
| employment_status | Employment type |
| annual_income | Customer's yearly income |
| credit_score | Creditworthiness score |
| risk_level | Low, Medium, High |

---

## merchants.csv

Contains merchant information.

| Column | Description |
|--------|-------------|
| merchant_id | Unique merchant identifier |
| merchant_name | Store name |
| category | Business category |
| city | Merchant city |
| country | Merchant country |
| commission_rate | Percentage earned by FinPay |
| rating | Merchant rating |
| active_status | Active or inactive |
